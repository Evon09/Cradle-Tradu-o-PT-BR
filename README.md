# Cradle - Tradução PT-BR

Este projeto contém a tradução em português do jogo *Cradle*.

## 📌 Sobre

Tradução feita de forma independente, com auxílio de ferramentas de IA.  
O objetivo é tornar o jogo mais acessível para jogadores brasileiros.

⚠️ Observações:
- Não sou fluente no idioma original.
- Não finalizei o jogo, então podem existir inconsistências.
- Evitei spoilers durante o processo.

---

## ⚙️ Instalação

Siga os passos abaixo:

1. Baixe este repositório (ou o arquivo de release).
2. Extraia os arquivos e coloque a pasta **Cradle_PT_BR** na raizd do jogo *Cradle*.
3. Execute o arquivo `.bat` incluído.
4. Aguarde o processo automático ser concluído.

---

## ⏱️ Tempo de execução

O processo pode levar alguns segundos ou alguns minutos (kkkk não cronometrei isso).
